Welcome to my portfolio website. I started coding little bits and bobs a few years ago but only recently started learning properly about code and web development.

I am currently working through the Codecademy FullStack Dev course. Here are some projects I have been working on.

Didis Unicorn Shop - Feb 23


My eldest daughter loves unicorns so when I asked asked to make a business landing page I decided to take inspiration from her. Her and her sister added some of their own content.

Two elements of the design that I focused on involved using media queries to make the nav responsive and the meet the team flexboxes switch at breakpoints.

Christmas Project - Mar 23


The brief was make something fun and Christmas themed for the local Codebar Christmas Competition.

I based this project around the song "The Twelve Days of Christmas", creating a calculator for the number of presents received each day and in total.

In hindsight it was quite an ambitious project so early in my coding journey but I sought help where needed, building and rebuilding to ensure I understood each part of the code I was using.
